Multi-Agent Education Assistant

> Multi-agent AI driven learning platform

Application Output

The Multi-Agent Education Assistant provides distinct interfaces for students and teachers, enabling real-time learning feedback, performance tracking, and AI-driven teaching interventions.

Students receive personalized feedback and can directly interact with an AI assistant, while teachers gain access to dashboards highlighting student performance trends and automated struggle alerts.

Screenshots of the main page, student dashboard, teacher dashboard, performance analytics, and AI tools will be displayed here.

Code & Architecture

This project is built using the Flask web framework and implements a multi-agent AI architecture that integrates OpenAI and Google Gemini models. Each model generates independent responses, which are then synthesized into a cohesive output aligned with Operating System Theory and Principles terminology.

  • Flask routing and session management
  • Secure API key handling via environment variables
  • Multi-agent response synthesis pipeline
  • Student performance logging and grading rubric
  • Teacher dashboards with automated alerts
